# docs

Architecture notes and decision records for Atlas v1.

## Structure

```
architecture/   # system design docs (RAG pipeline, data flow, component diagrams)
adr/            # Architecture Decision Records (numbered, immutable once accepted)
```

## Conventions

- ADRs follow the standard format: Context, Decision, Consequences.
- Name ADR files `NNNN-short-title.md` (e.g. `0001-vector-store-choice.md`).

## Status

Scaffold only — first ADRs to be added as v1 design decisions are made.
